# Quintana Concrete, LLC

Official website for **Quintana Concrete, LLC** — Arizona's premier concrete specialists serving the Greater Phoenix Area.

Built with [Next.js](https://nextjs.org/) and deployed on [Vercel](https://vercel.com/).

## Services

- Concrete Driveways
- Pathways & Sidewalks
- Custom Slabs & Forming
- Commercial & Residential Concrete Work

## Contact

**Phone:** (602) 471-3046
**Service Area:** Greater Phoenix Area & Surrounding Cities

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site locally.

## Build

```bash
npm run build
npm start
```

## Deployment

This project is automatically deployed to Vercel on every push to `main`.
